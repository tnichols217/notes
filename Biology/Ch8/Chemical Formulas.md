# #ATP and #ADP
- ATP is a Hydrogen and Oxygen carrier
- The base is ADP
- ADP + **1H + 3O + 1P** = ATP

# #NADH and #NAD<sup>+</sup>
- NADH is a hydrogen carrier, it carries **2H**

# #NADPH and #NADP
- NADPH is a hydrogen carrier, it carries **1H**

# #FADH and #FAD
- FADH is a hydrogen carrier, it carries **2H**
