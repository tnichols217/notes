# Useful Identities

## The Gamma Function

$\Gamma(\alpha)=\int\limits_0^\infty t^{\alpha-1}e^{-t}\qquad \text{for }\alpha>0$

- $\Gamma(\alpha)=(\alpha-1)\Gamma(\alpha-1)$
- $\Gamma(0.5)=\sqrt\pi$

## The Exponential Function

$e^\lambda=\sum\limits_{x=0}^\infty\frac{\lambda^x}{x!}$
