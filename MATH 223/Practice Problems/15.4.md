# 7

$\int\limits_{-2}^2\int\limits_0^{\sqrt{4-x^2}}x^2+y^2~dydx$


```function-plot
{
	data: [
		{
			fn: "sqrt(4-x^2)",
			closed: true,
		}
	],
	xAxis: {
		domain: [-3, 3]
	},
	yAxis: {
		domain: [-1, 3]
	}
}
```

$0<r<2$
$0<\theta<\pi$

$\int\limits_0^\pi\int\limits_0^2r^2r~drd\theta$
$=\int\limits_0^\pi\bigg|_0^2r^4/4~drd\theta$
$=\int\limits_0^\pi4~d\theta$
$=4\pi$
