# Sorting

## Problem Definition

**Input**:

```java
Array A = <a1, a2, a3, ..., an>
```

**Output**:

```java
Array B = <a1'', a2'', a3'', ..., an''>
```

Such that

- `a1 <= a2 <= a3 <= an`
- All elements in `A` are in `B`
