# Units
A unit is a scale of which a measurement is made in

All answers and measurements should have units, with a few minor exceptions (like ratios)

## SI units
- mass: kg
- length: m
- time: s
- temperature: K
- substance: mol
- electrical charge: J

## Metric Prefixes
| name | prefix | amount |
| ---- | ------ | ------ |
| mega | M      | $10^6$ |
| kilo | k      | $10^$       |
